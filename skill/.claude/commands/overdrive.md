# /overdrive
Push the target to maximum visual impact — bold typography, dramatic contrast, cinematic layout, and unapologetic design presence.

## When to use
- When a landing page, hero section, or marketing page needs to make a statement
- When the design is too safe and needs to stand out
- When you want to explore the upper bound of visual intensity

## Steps
1. **Identify the hero moment** — What is the single most important thing on this page? Everything else serves it.
2. **Scale up** — Increase heading sizes dramatically. Use the largest sizes in the type scale. Let the hero content breathe with generous whitespace.
3. **Contrast** — Push contrast to maximum. Dark backgrounds with light text, or vice versa. Use color boldly — full saturation accents against neutral backgrounds.
4. **Simplify ruthlessly** — In overdrive mode, less is more. Remove anything that doesn't amplify the core message. Every element must earn its place.
5. **Motion with purpose** — Add entrance animations, parallax, or scroll-triggered reveals. Motion should build anticipation and guide the narrative.
6. **Check the system** — Even in overdrive, use design system tokens and components. Bold doesn't mean breaking the system — it means using it at full volume.

## Output
- High-impact code with bold design choices applied
- Notes on what was amplified and what was removed
- Warning if any changes break system compliance
