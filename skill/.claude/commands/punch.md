# /punch
Increase the visual intensity of the target — stronger colors, heavier weights, tighter spacing, more contrast. Make it hit harder.

## When to use
- When a UI feels too flat or passive
- When CTAs don't stand out enough
- When important information gets lost in the page
- When a dashboard needs to surface key metrics more prominently

## Steps
1. **Identify what needs emphasis** — What should be louder? CTAs, key metrics, status indicators, headings?
2. **Color up** — Shift weak colors to their stronger variants (e.g., `text-neutral-weak` → `text-neutral-default`). Add color to previously neutral elements where it adds meaning.
3. **Weight up** — Increase font weights on key elements. Regular → semibold. Medium → bold. Make headings command attention.
4. **Contrast up** — Increase the contrast between foreground and background. Ensure interactive elements pop against their context.
5. **Tighten spacing** — Reduce excessive whitespace that dilutes impact. Group related elements closer together.
6. **Sharpen interactions** — Make hover states more distinct, focus rings more visible, active states more pronounced.

## Output
- Punched-up code with increased intensity
- Summary of what was amplified
